## 🐾 波妞消消樂 (波弟與波妞的冒險遊戲) 🐱✨

### 引言：在指尖感受貓咪的萌力與連鎖消除的快感

誰能抗拒可愛的貓咪波弟與萌萌的小女生波妞呢？**波妞消消樂 (Match-3 Game)** 是一款專為療癒心靈而設計的網頁版三消益智遊戲。我們將充滿貓咪元素的圖形（如貓臉、肉球 🐾、毛線球等）融入經典的消除玩法中。

透過高效能的 Canvas 渲染與流暢的動畫，這款遊戲不僅在視覺上充滿活力，更在操作感上追求極致的「絲滑」。無論您是想在閒暇時放鬆一下，還是想挑戰手速極限，這裡都有屬於您的冒險。

--------------------------------------------------------------------------------

### 🎮 遊戲特點：雙重模式，雙倍樂趣

為了滿足不同類型的玩家，遊戲設計了兩套截然不同的挑戰目標：

*   **🧘‍♂️ 經典模式**：無時間壓力，專注於尋找最佳消除路徑，享受連鎖爆發帶來的視覺與聽覺饗宴。
*   **🔥 計時模式**：在緊張的 60 秒內挑戰最高分，每一秒的遲疑都是分數的流失，測試您的極限反應。

#### 🌟 獨家功能
*   **📱 觸控深度優化**：支援直覺的 **Swipe (滑動)** 手勢，指哪換哪，專為行動裝置量身打造。
*   **🎵 動態音效回饋**：音效會隨著連鎖消除次數（Combo）自動提高音調，增加遊戲的緊張感與成就感。
*   **🎬 影片即時慶祝**：畫面下方內建波弟與波妞的慶祝影片，平時安靜陪玩，一旦觸合連鎖，則會瞬間跳轉至興奮舞動的高潮片段！🐱💥

--------------------------------------------------------------------------------

### 💡 技術挑戰與優化：打造 60 FPS 的流暢體驗

在開發過程中，我們解決了多項針對行動端的效能與排版難題：

1.  **🖼️ 棋盤自動適應**：透過自研的寬度優先渲染邏輯，確保在不同比例的螢幕上棋盤都能呈現「滿版」的飽滿視覺，移除紅色空白區域。
2.  **🎥 影片狀態機**：使用 JavaScript 精準管理 `cheer.mp4` 的播放時間點 (timeupdate)，實現遊戲邏輯與影片慶祝的完美同步。
3.  **⚖️ UI 現代化**：採用 Flexbox 整合標題與模式切換區塊，並統一霓虹發光效果，讓介面既專業又具現代感。

--------------------------------------------------------------------------------

### 🛠️ 技術棧揭秘

*   **核心引擎**：Vanilla JavaScript (ES Modules)，保證最輕量的執行負擔。
*   **渲染技術**：HTML5 Canvas API，實現高頻率的方塊位移與粒子效果。
*   **樣式系統**：CSS3 動態漸層與 RWD 響應式佈局。
*   **雲端託管**：透過 Cloudflare Pages 實現全球毫秒級發布。
