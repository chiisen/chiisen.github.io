## 🚀 數學怪物射手 (Math Monsters Shooter)

### 引言：當數學練習變成一場太空冒險

對於剛接觸加減法的國小低年級學生來說，枯燥的習題往往讓人望而生畏。**數學怪物射手 (Math Monsters Shooter)** 的誕生，就是為了解決這個痛點——將繁瑣的數學計算轉化為緊張刺激的射擊遊戲，讓孩子在保衛銀河系的過程中，不知不覺提升運算能力與反應速度。

透過現代 Web 技術，這款遊戲提供了一個流暢且具備視覺衝擊力的「霓虹太空」世界，讓學習不再是苦差事。

--------------------------------------------------------------------------------

### 🎮 遊戲與運算模式

遊戲提供了多樣化的選擇，以適應不同學習階段的需求：

1.  **加法模式 (Addition)**：強化基礎與進階的加法練習。
2.  **減法模式 (Subtraction)**：訓練逆向運算的敏銳度。
3.  **綜合挑戰 (Mix Mode)**：加減混合，全面鍛煉大腦的切換反應。

#### ⭐ 難度選擇
*   **Easy (單個數字)**：適合初學者建立信心。
*   **Medium (1-10)**：穩定的基礎功訓練。
*   **Hard (10-20)**：挑戰更高階的進位與借位運算。

--------------------------------------------------------------------------------

### 🎨 視覺與音效體驗：Neon Space 風格

MiniOC 秉持著「極簡且安全」的設計目標，確保即便是在行動端，也能享有快速、穩定且受保護的互動體驗。

*   **主題色彩**：以深邃藍 (#0B0F29) 為星際背景，搭配霓虹青 (#00F3FF) 與洋紅 (#FF00FF) 的子彈與怪物效果。
*   **特寫效果**：內建精心調校的粒子爆炸系統、物件發光 (Glow) 與毛玻璃 (Glassmorphism) 介面設計。
*   **動態音效**：使用 Web Audio API 實現零依賴的高品質音效，增強遊戲的沉浸感。

--------------------------------------------------------------------------------

### 🛠️ 技術架構：高效能的瀏覽器遊戲

這款遊戲雖然看起來輕巧，但背後使用了現代前端技術棧來確保在流動裝置與電腦上都能流暢執行：

*   **Core Framework**：Vue 3 + Vite 確保開發效率與快速加載。
*   **Rendering ENGINE**：使用原生 **HTML5 Canvas** 進行渲染，確保在高頻率粒子生成時仍能保持 60fps。
*   **State Management**：利用 Vue 的 Reactivity API 進行即時的遊戲狀態同步。
*   **Deployment**：部署於 Cloudflare Workers / Pages，提供極速的內容分發。
