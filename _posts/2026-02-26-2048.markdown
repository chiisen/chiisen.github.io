## 🎮 2048 網頁版：滑動、合併、挑戰極限
![2048](/image/github.io/2048.png)

### 引言：簡單數字，無窮樂趣

自 2014 年橫空出世以來，2048 以其極簡的規則與深奧的策略，成為了無數人殺時間的首選。**2048 網頁版 (2048 Web)** 重新實現了這款經典遊戲，致力於在瀏覽器中提供最流暢、最具回饋感的遊戲操作。

無論您是為了打發通勤時間，還是想要挑戰 4096 甚至更高的極限，這款針對行動端優化的版本都能滿足您的需求。

--------------------------------------------------------------------------------

### 🌟 功能特性：優雅視覺與靈魂互動

*   **✨ 流暢動畫**：透過精心調校的 CSS Transition，方塊的移動與合併呈現極致的平滑感。
*   **💥 動態回饋**：當方塊成功合併時，具備動態的縮放彈跳效果，讓每一次合成都充滿成就感。
*   **🎨 視覺化進度**：方塊顏色隨數字增加而加深，從淺色調過渡到充滿力量的深色調，視覺化呈現您的挑戰進度。
*   **🎉 勝利慶祝**：達成 2048 目標時，內建 `canvas-confetti` 噴泉花絮效果，為您的智慧喝采。
*   **📱 響應式佈局**：RWD 設計確保在不同尺寸的螢幕上都能展現完美的 4x4 棋盤比例。

--------------------------------------------------------------------------------

### 🕹️ 遊戲玩法與操控：指尖上的博弈

目標非常簡單：滑動方塊，將相同數字的方塊合併為兩倍數值，直到拼湊出 2048！

1.  **💻 電腦玩家**：直接使用鍵盤方向鍵 `↑` `↓` `←` `→` 控制方塊滑動方向。
2.  **📱 手機玩家**：在遊戲區域直接向上下左右滑動，手勢指令將無縫轉化為動作。

#### 核心規則：
*   每次移動後，棋盤會隨機生成一個新方塊（2 或 4）。
*   當所有格子被填滿且無法進行任何合併移動時，遊戲結束。

--------------------------------------------------------------------------------

### 🛠️ 技術架構：輕量且不失質感

*   **Vanilla JS**：核心遊戲邏輯使用純 JavaScript 編寫，確保極低的運算負擔。
*   **CSS Transition & Flexbox**：打造響應式網格與高品質的補間動畫。
*   **Fredoka Font**：選用圓潤友好的字體，提升整體視覺質感。
*   **Cloudflare Workers**：透過無伺服器架構提供全球極速載入與託管。
