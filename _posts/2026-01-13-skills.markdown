## Claude Skills：讓 AI 從「說話」進化到「動手做」的擴展能力

![Claude_MCP](/image/github.io/Claude_MCP.gif)

### 引言：AI 不再只是「聊天機器人」，而是你的「數位專員」

在過去的兩年中，我們習慣了與 Large Language Models (LLMs) 進行對話，獲取建議或生成文本。然而，Anthropic 的 Claude 3.5 Sonnet 正帶領我們進入下一個階段：**Claude Skills**。這不再僅限於文字生成，而是透過「工具調用 (Tool Use)」與外部世界互動。

其核心邏輯在於：「當模型發現自己無法獨立完成任務時，主動要求執行特定的程式碼或 API。」這讓 AI 具備了真正的操作能力，成為不僅能思考、還能執行的數位專員。

--------------------------------------------------------------------------------

### 核心概念：什麼是 Claude Skills？

Claude Skills 的強大建立在三個核心支柱之上：

#### 1. 工具調用 (Tool Use / Function Calling)
這是 Skills 的基石。開發者定義好函數的名稱、描述與參數格式（JSON Schema），Claude 根據對話語境判斷何時需要調用該工具。這就像是賦予 AI 一個工具箱，讓它在需要時主動拿出正確的工具（如：搜尋網頁、計算數據）。

#### 2. 電腦操作能力 (Computer Use)
這是目前最前沿的突破。模型可以直接「觀察」螢幕截圖、移動滑鼠、點擊按鈕、輸入文字。這讓 Claude 能夠像人類一樣操作任何現有的軟體（如：Excel、瀏覽器、IDE），解決了許多軟體本身沒有提供 API 的困境。

#### 3. 模型上下文協定 (Model Context Protocol, MCP)
MCP 是 Anthropic 推出的開放標準。它讓開發者能建立一個通用的「服務器」，將本地數據（如資料庫、檔案系統）或遠端服務封裝，讓 AI 即時掛載並使用這些能力。

--------------------------------------------------------------------------------

### 精選技能清單：AI 助理的「新天賦」

目前 [chiisen/skills](https://github.com/chiisen/skills) 專案維護了多項專用技能，將 AI 的戰力擴展至各個領域：

#### 🛠️ 開發與程式碼審查 (Development)
*   **[Code Reviewer](https://github.com/chiisen/skills/blob/main/code-reviewer/README.md)**：專業的審核專家，針對變更範疇、風格、安全與測試提供建設性回饋。
*   **[Semantic Git](https://github.com/chiisen/skills/blob/main/semantic-git/SKILL.md)**：根據 `git diff` 自動生成符合 Conventional Commits 規範的訊息。

#### 🩺 環境診斷與修復 (Infrastructure)
*   **[Debug Wizard](https://github.com/chiisen/skills/blob/main/debug-wizard/SKILL.md)**：自動執行容器狀態巡檢與關鍵端口連通性測試。
*   **[Log Sentinel](https://github.com/chiisen/skills/blob/main/log-sentinel/SKILL.md)**：快速撈取 Docker 日誌並自動濾除雜訊，僅保留關鍵錯誤資訊。
*   **[Zsh Fix](https://github.com/chiisen/skills/blob/main/zsh-fix/README.md)**：自動診斷並修復終端機歷史紀錄遺失或外掛失效的問題。

#### 📝 筆記與視覺化協作 (Knowledge)
*   **[Obsidian Markdown & Bases](https://github.com/chiisen/skills/blob/main/obsidian-markdown/README.md)**：完美支援雙向連結、提示框與動態資料庫視圖。
*   **[JSON Canvas](https://github.com/chiisen/skills/blob/main/json-canvas/README.md)**：直接處理視覺化畫布，製作心智圖與流程圖。

--------------------------------------------------------------------------------

### 重點摘要：技術預研的核心觀察

1.  **從被動到主動 (Passive to Active)**：具備 Skills 的 Claude 是根據目標（Goal）去尋找工具，並自行決定執行的先後順序，不再只是被動回話。
2.  **環境感知 (Environmental Awareness)**：Computer Use 讓 Claude 具備操作作業系統 GUI 的能力，這對於環境調適 (Environment Setup) 是革命性的進步。
3.  **循環反饋機制 (Loop Feedback)**：Claude 執行後會觀察結果，如果失敗，它會自動修正參數並重新嘗試，直到達成任務。

--------------------------------------------------------------------------------

### 結語：跨越文字，擁抱行動

Claude Skills 代表了 AI 發展的一個重要分水嶺。當 AI 能夠「動手」處理瑣碎的部署作業、分析複雜的日誌或是整理龐雜的筆記時，人類開發者才能真正解脫出來，專注於更高層次的創意與架構設計。

未來，我們評估一個 AI 的標準，將不再只是「它對問題的回答有多精彩」，而是「它為我們解決了多少實際的工作流程」。
